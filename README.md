# AI-Supported_Unity_Game
AI-supported educational Unity game developed with C# and Unity.

An AI-supported educational game developed with Unity and C#. The project integrates Large Language Models (LLMs) to provide interactive NPC conversations, voice responses, and educational gameplay.

## Features

- 🤖 AI-powered NPC interaction using Gemini API
- 🔊 Text-to-Speech (TTS) support for NPC responses
- 💬 Real-time conversational gameplay
- ❓ Interactive question-answer system
- 🎮 User-friendly interface developed with Unity UI
- 🧠 Educational game mechanics
- 🏗️ Object-Oriented Programming (OOP) architecture

## Technologies

- Unity
- C#
- TextMeshPro
- Gemini API
- Google Text-to-Speech (TTS)
