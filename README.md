# 🎅 AI-Supported Unity Game

An AI-powered educational game developed with **Unity** and **C#**, featuring an interactive NPC that communicates with players using **Google Gemini AI**. The NPC answers questions in character, translates responses into English, and generates natural speech through Text-to-Speech technology.

## 📖 Overview

This project was developed to explore the integration of **Artificial Intelligence** into interactive game development.

The game features a Santa Claus NPC that remains in character throughout the conversation. Players can ask questions through the game interface, receive AI-generated responses, and hear the answers spoken aloud using Text-to-Speech.

The project demonstrates the integration of modern AI services with Unity while following object-oriented programming principles.

## ✨ Features

- 🤖 AI-powered NPC conversations using Google Gemini
- 🎅 Santa Claus role-playing system with custom prompts
- 💬 Real-time question & answer interaction
- 🌍 Automatic Turkish → English translation
- 🔊 Text-to-Speech (TTS) voice output
- 🎮 Interactive Unity UI
- 🧠 Object-Oriented Programming (OOP) architecture
- ⚡ REST API communication with JSON
- 📝 TextMeshPro-based user interface

## 🎮 Gameplay

The player interacts with Santa Claus through a chat interface.

1. The player enters a question.
2. The question is sent to the Gemini API.
3. Santa Claus responds while staying in character.
4. The response is translated into English.
5. The translated response is spoken using Text-to-Speech.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/busetas/AI-Supported_Unity_Game.git
```

2. Open the project using **Unity Hub**.

3. Open the main scene.

4. Configure your own **Gemini API Key**.

5. Press **Play**.


## 🔑 API Configuration

This project requires a valid **Google Gemini API Key**.

For security reasons, API keys are **not included** in this repository.

Create your own API key and configure it inside the project before running the game.
